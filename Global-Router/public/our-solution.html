News Desserts

Have become an increasingly prevolent issue across the United States and the globe. According to Medill, 55 million Americans are currently living in regions with limited local news coverage.

The Solution,

By creating a platform to connect local journalists with the community, we aim to help fill the local news gap by providing journalists credible sources they can use to optimize local coverage.

AI Validity Checks

We use AI to check the validity of posts, ensuring that information shared is credible and trustworthy.

Source Citation & Mentions

Provide options to cite sources by mentioning people in your stories and including image and video material. Whenever a person is mentioned, the system will automatically notify them and allow them to validate the claim.

Downloadable Formats

We allow users to download stories in multiple data formats including JSON, CSV, Markdown, and raw text files.

Our Tech Stack: How It Works

We use PyTorch alongside FastAPI to run an efficient AI server capable of analyzing claims. Our main global router is built with Node.js and Express, which routes users to the appropriate server distribution for their region.

Specifically, each covered region has a dedicated Docker volume and SQL server. When a user connects, the global router checks if a volume exists for their region. If it does, it then checks if a server is active for that region. If not, it pulls up an idle server from our inactive server base and uses Kubernetes to integrate it with the database, efficiently managing resources and ensuring seamless scalability.
